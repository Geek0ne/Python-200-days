## ✍️ 练习题

### 练习 1：变量交换

不使用第三个变量，交换两个变量的值。

```python
a = 10
b = 20
# 请在这里写你的代码，使得 a=20, b=10

# 验证
print(f"a = {a}, b = {b}")  # 应输出 a = 20, b = 10
```

**提示**：Python 特有的元组解包方式是什么？

### 练习 2：类型侦探

写一个程序，给定任意输入，判断并输出它属于哪种类型：

```python
# 给以下每个值打印出它的类型
items = [42, 3.14, "3.14", True, None, "True", 0, "", 0.0, 2**100]

# 对每个 item，输出 item, type(item).__name__

# 额外挑战：说出哪些值在 bool() 转换后为 False
```

### 练习 3：格式化输出练习

创建一个程序，定义以下变量并使用 f-string 输出格式化的自我介绍：

```python
name = "你的名字"          # 替换为你的名字
age = 你的年龄              # 替换为你的年龄
dream_job = "你的理想职业"  # 替换为你的理想职业

# 用 f-string 输出： "大家好，我是[name]，今年[age]岁，我的梦想是成为一名[dream_job]。"
# 要求：年龄后面的"岁"字也是变量拼进去的
# 输出示例： "大家好，我是张三，今年25岁，我的梦想是成为一名宇航员。"
```

### 练习 4：`is` 与 `==` 的实验

# 1. 找出两个不同的变量，使得 a is b 为 True 但 a == b 也为 True

### 练习 5：个人信息收集器升级版
