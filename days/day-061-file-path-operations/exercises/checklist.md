## 💡 思考题答案提示

1. **100 万文件统计**：用 `Path.rglob("*.log")` 配合生成器逐个处理，避免一次性加载所有文件到内存。`os.walk` 也可以，但 pathlib 的 API 更清晰。

2. **清理工具设计**：用 `Path.rglob("*.log")` 查找日志，检查 `stat().st_mtime` 判断时间，`shutil.copy2` + `shutil.make_archive` 压缩，`Path.unlink` 删除。

3. **copy vs copy2**：`copy` 只复制内容和权限，`copy2` 还复制元数据（时间戳等）。备份场景必须用 `copy2`，否则无法判断文件是否更新。

4. **安全删除函数**：先 `p.exists()` 检查，`p.is_symlink()` 跳过符号链接，检查 `p.resolve()` 是否在白名单目录内，最后 `p.unlink()`。

5. **os.listdir vs Path.iterdir**：`os.listdir` 返回字符串列表，内存开销小；`Path.iterdir` 返回 Path 对象，功能更丰富但有对象创建开销。处理大量文件时 `os.listdir` 略快。
