## 📝 基础练习

### 练习 1：浅拷贝陷阱
创建一个嵌套列表 `data = [[1, 2], [3, 4]]`，使用浅拷贝创建副本，然后修改副本的嵌套元素，验证原列表是否被修改。

```python
# 预期输出:
# 修改后原始列表: [[1, 2], [3, 4, 5]]
# 嵌套对象被共享了！
```

### 练习 2：深拷贝验证
创建一个包含循环引用的对象，使用深拷贝创建副本，验证：
1. 副本与原对象是独立的
2. 循环引用关系被正确复制
3. 修改副本不影响原对象

### 练习 3：不可变对象拷贝
测试以下对象的拷贝行为：
- 整数 `42`
- 字符串 `"hello"`
- 元组 `(1, 2, [3, 4])`
- 元组 `(1, 2, 3)`

解释为什么有些返回原对象，有些返回新对象。
