## 4. 去重指纹生成流程

```
Request(url, method, body)
    │
    ▼
┌──────────────────────────────┐
│  1. 提取 URL 参数             │
│  url = "https://a.com/p=1&t=2"│
│  params = {p: [1], t: [2]}   │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│  2. 移除忽略参数（可选）       │
│  remove(params, 't')         │
│  params = {p: [1]}           │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│  3. 排序并重建 URL            │
│  clean_url = "https://a.com/p=1"│
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│  4. 生成哈希指纹             │
│  fp = sha1(clean_url + method + body)│
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│  5. 检查指纹集合              │
│  fp in seen?                  │
│  ├─ No  → 允许，记录指纹      │
│  └─ Yes → 过滤掉              │
└──────────────────────────────┘
```

## 5. 断点续爬机制

```
第一次运行                          中断后恢复
    │                                   │
    ▼                                   ▼
┌─────────────┐                   ┌─────────────┐
│ 开始爬取     │                   │ 读取 JOBDIR  │
│             │                   │             │
│ 处理 50%    │     进程崩溃       │ 恢复队列状态 │
│    │        │ ─────────────→    │ 恢复去重集合 │
│    ▼        │                   │    │        │
│ 保存状态    │                   │    ▼        │
│ 到 JOBDIR   │                   │ 继续爬取     │
│             │                   │ 剩余 50%    │
└─────────────┘                   └─────────────┘

JOBDIR 保存的内容：
├── queue/          # 请求队列（未处理的请求）
├── dupefilter/     # 去重指纹集合
└── requests.queue  # 调度器状态
```
