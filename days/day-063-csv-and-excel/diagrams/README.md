# Day 063 — CSV 与 Excel：图解

## 1. CSV 数据流

```
CSV 文件 (.csv)                   内存中 Python 对象
┌──────────────────┐            ┌──────────────────────┐
│ name,age,city    │  reader()  │ ['name', 'age', 'city']│
│ Alice,30,Beijing │ ────────→ │ ['Alice', '30', 'Beijing']│
│ Bob,25,Shanghai  │            │ ['Bob', '25', 'Shanghai']│
│ Charlie,35,SZ    │            │ ['Charlie', '35', 'SZ']│
└──────────────────┘            └──────────────────────┘
         │                               │
         │ DictReader()            DictWriter()
         ▼                               ▼
┌──────────────────┐            ┌──────────────────────┐
│ [{'name':'Alice',│            │ 字符串 → 数字转换     │
│   'age':'30',    │            │ json.loads → 类型推导 │
│   'city':'Beijing'}, ...]     │ 日期解析 → datetime   │
└──────────────────┘            └──────────────────────┘
```

## 3. 数据读取路径对比

```
                    CSV / Excel 数据
                           │
              ┌────────────┴────────────┐
              │                         │
      csv 模块 / openpyxl           pandas
              │                         │
              ▼                         ▼
┌──────────────────────┐   ┌──────────────────────┐
│ 逐行读取             │   │ 批量加载到内存        │
│ 低内存消耗           │   │ 自动类型推断          │
│ 手动类型转换         │   │ 丰富的数据操作 API    │
│                      │   │                      │
│ ✅ 适合超大文件      │   │ ✅ 适合数据分析       │

│ ❌ 无类型推断        │   │ ❌ 依赖重             │
└──────────────────────┘   └──────────────────────┘
```

## 4. CSV 解析状态机

编写正确的 CSV 解析器比看起来复杂得多：

```
                 ┌─────┐
                 │ 开始 │
                 └──┬──┘
                    │ 读取字符
                    ▼
              ┌───────────┐
          ┌──→│ 字段开始   │←────────────────────┐
          │   └─────┬─────┘                      │
          │         │ 遇到引号?                   │
          │    ┌────┴────┐                       │
          │    │ 是      │ 否                     │
          │    ▼         ▼                       │
          │ ┌──────┐ ┌──────┐                    │
          │ │引号内 │ │普通  │                    │
          │ └──┬───┘ └──┬───┘                    │
          │    │        │                        │
          │    │ 双引号? │ 分隔符?                │
          │    ├──是──┐  ├──是──→ 字段结束        │
          │    │     │  │                        │
          │    │ 否  │  │ 换行?                  │
          │    │     │  ├──是──→ 记录结束         │
          │    ▼     │  │                        │
          │ ┌──────┐ │  │                        │
          │ │引号结束│←┘  │                        │
          │ └──┬───┘    │                        │
          │    │        │                        │
          └────┘────────┘                        │
                                                │
                 ┌──────────────────────────────┘
                 │ 遇到 escapechar?
                 ▼
          ┌───────────┐
          │ 转义处理   │
          └───────────┘
```

## 5. Excel 样式体系

```
Excel 单元格
│
├── Font (字体)
│   ├── name='Microsoft YaHei'
│   ├── size=12
│   ├── bold=True
│   ├── color='4472C4'
│   └── italic=False
│
├── Fill (填充)
│   ├── PatternFill(start_color='4472C4', fill_type='solid')
│   └── GradientFill(...)
│
├── Alignment (对齐)
│   ├── horizontal='center'
│   ├── vertical='center'
│   └── wrap_text=True
│
├── Border (边框)
│   ├── left=Side(style='thin', color='D9E2F3')
│   ├── right=Side(style='thin')
│   ├── top=Side(style='thin')
│   └── bottom=Side(style='thin')
│
├── Number Format (数字格式)
│   ├── '#,##0'         → 千分位整数
│   ├── '¥#,##0.00'     → 货币格式
│   ├── '0.00%'         → 百分比
│   └── 'YYYY-MM-DD'    → 日期格式
│
└── Protection (保护)
    └── locked=True/False
```
