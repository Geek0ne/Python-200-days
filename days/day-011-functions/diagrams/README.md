## 3. 参数传递机制详解

### 不可变类型（int）

```
调用前:                        调用中:                    调用后:
n = 5                         ┌───────────────┐          n = 5
n ───→ [int: 5]               │ x ──→ [int:5] │          n ───→ [int: 5]
                               │               │
                               │ x = 100       │
                               │ x ──→ [int:100]│
                               └───────────────┘
```

### 可变类型（list）—— 修改内容

```
调用前:                        调用中:                    调用后:
my_list = [1,2,3]             ┌──────────────────┐       my_list = [1,2,3,4]
my_list ──→ [1,2,3]           │ lst ──→ [1,2,3]  │       my_list ──→ [1,2,3,4]
                               │           │      │
                               │     lst.append(4)  │
                               │           │      │
                               │           ▼      │
                               │         [1,2,3,4] │
                               └──────────────────┘
```

### 重新绑定 vs 修改对象

```
重新绑定 (不影响外部):               修改对象 (影响外部):

外部:                      外部:
[1, 2, 3]                  [1, 2, 3]
    ↑                          ↑
    │                          │
lst (参数) ──→ 断开了!     lst (参数) ──→ 仍然指向同一个
    │                          │
    ▼                          ▼
[100, 200, 300]            [1, 2, 3, 999]
(新对象，外部再也看不到了)    (外部能感知到这个修改)
```

## 4. return 的执行流程

```
函数调用
    │
    ▼
┌──────────────────┐
│  执行函数体        │
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│  遇到 return 语句  │
│       或          │
│  函数体执行完毕     │
└──────┬───────────┘
       │
       ▼
┌──────────────────────────────────┐
│  计算 return 后面的表达式（如果有） │
│  return a + b                    │
│         ↓                        │
│     计算 a + b → 得到结果         │
└──────────────┬───────────────────┘
               │
               ▼
┌──────────────────┐
│  销毁当前栈帧      │  ← 局部变量消失
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│  返回值传递给调用者  │
└──────────────────┘
```

## 8. 牛顿迭代法求平方根（可视化）

```
求 sqrt(10)

第一次迭代: x₀ = 10
    切线: f(x) = x² - 10
    x₁ = (10 + 10/10) / 2 = 5.5

第二次迭代: x₁ = 5.5
    x₂ = (5.5 + 10/5.5) / 2 = 3.66

第三次迭代: x₂ = 3.66
    x₃ = (3.66 + 10/3.66) / 2 = 3.20

第四次迭代: x₃ = 3.20
    x₄ = (3.20 + 10/3.20) / 2 = 3.16

第五次迭代: x₄ = 3.16
    x₅ = (3.16 + 10/3.16) / 2 = 3.1623 ✅

收敛过程:
10.00 → 5.50 → 3.66 → 3.20 → 3.16 → 3.1623 (实际: 3.16227766)
```

## 10. 总结：函数的核心概念关系图

```mermaid
flowchart TD
    DEF[def 关键字定义函数] --> PARAM[参数列表]
    DEF --> BODY[函数体]
    BODY --> DOC[文档字符串 docstring]
    BODY --> CODE[执行代码]
    BODY --> RET[return 返回值]
    
    PARAM --> IMMUT[不可变: int/str/tuple]
    PARAM --> MUT[可变: list/dict/set]
    
    IMMUT --> VAL[值传递错觉<br/>重新绑定不影响外部]
    MUT --> REF[引用传递<br/>修改内容影响外部]
    
    RET --> SINGLE[单个值]
    RET --> MULTI[多个值 -> 元组]
    RET --> NONE[没有return -> None]
    
    DEF --> CITIZEN[一等公民 First-class]
    CITIZEN --> ASSIGN[赋值给变量]
    CITIZEN --> PASS[作为参数传递]
    CITIZEN --> RETURNED[作为返回值]
    
    DEF --> PRINCIPLE[设计原则]
    PRINCIPLE --> SRP[单一职责原则]
    PRINCIPLE --> NAMING[命名规范]
    PRINCIPLE --> DOC_STR[写 docstring]
    
    style DEF fill:#ff9800,color:#fff
    style BODY fill:#4caf50,color:#fff
    style RET fill:#f44336,color:#fff
    style CITIZEN fill:#9c27b0,color:#fff
```
