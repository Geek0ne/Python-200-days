### 练习 5: 函数分析器

**要求：**
编写一个函数 `analyze_function(func, test_inputs)`，它能：

1. 接收一个函数和一组测试输入
2. 对每个输入调用函数
3. 返回一个包含以下信息的字典：
   - `function_name`: 函数名
   - `docstring`: 文档字符串
   - `results`: 输入 → 输出 的映射
   - `total_time`: 总执行时间（用 `time.time()` 测量）

**示例：**
```python
def square(x):
    """返回 x 的平方。"""
    return x * x

analysis = analyze_function(square, [1, 2, 3, 4, 5])
print(analysis)
# {
#   'function_name': 'square',
#   'docstring': '返回 x 的平方。',
#   'results': {1: 1, 2: 4, 3: 9, 4: 16, 5: 25},
#   'total_time': 0.000123  # 示例值
# }
```
