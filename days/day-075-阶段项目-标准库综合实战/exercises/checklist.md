# Day 075 练习题清单

## 练习 1：扩展功能 ⭐⭐
为重命名工具添加以下功能：
- 按文件大小重命名
- 按文件类型分组重命名
- 批量移动到子目录
- 支持撤销操作

**要求：**
- 实现 `size_rename()` 方法
- 实现 `group_by_type()` 方法
- 实现 `move_to_subdirs()` 方法
- 实现 `undo()` 方法（基于 JSON 报告）
- 编写完整的测试用例

**要求：**
- 实现文件上传功能
