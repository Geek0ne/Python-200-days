## 3️⃣ 冒泡排序过程

```
原始: [5, 3, 8, 6, 4]

第1趟:
  5 vs 3 → 交换 [3, 5, 8, 6, 4]
  5 vs 8 → 不动 [3, 5, 8, 6, 4]
  8 vs 6 → 交换 [3, 5, 6, 8, 4]
  8 vs 4 → 交换 [3, 5, 6, 4, 8]   ← 8 冒泡到位

第2趟:
  3 vs 5 → 不动 [3, 5, 6, 4, 8]
  5 vs 6 → 不动 [3, 5, 6, 4, 8]
  6 vs 4 → 交换 [3, 5, 4, 6, 8]   ← 6 冒泡到位

第3趟:
  3 vs 5 → 不动 [3, 5, 4, 6, 8]
  5 vs 4 → 交换 [3, 4, 5, 6, 8]   ← 5 冒泡到位

第4趟:
  3 vs 4 → 不动 [3, 4, 5, 6, 8]   ← 全部有序
```

## 4️⃣ 选择排序过程

```
原始: [5 | 3, 8, 6, 4]
       ↑已排序   ↑未排序区

第1趟: 找未排序最小值 3
       [3 | 5, 8, 6, 4]   ← 3 与 5 交换

第2趟: 找未排序最小值 4
       [3, 4 | 8, 6, 5]   ← 4 与 5 交换

第3趟: 找未排序最小值 5
       [3, 4, 5 | 6, 8]   ← 5 与 8 交换

第4趟: 找未排序最小值 6
       [3, 4, 5, 6 | 8]   ← 不动
```

## 5️⃣ 插入排序过程

```
初始:  [5 | 3, 8, 6, 4]
       ↑已排序 ↑未排序

第1步: 拿出 3, 与 5 比较
       [5, 5, 8, 6, 4]   → 5 > 3, 右移
       [3, 5, 8, 6, 4]   → 插入 3

第2步: 拿出 8, 与 5 比较
       [3, 5, 8, 6, 4]   → 8 > 5, 不动

第3步: 拿出 6, 与 8 比较
       [3, 5, 8, 8, 4]   → 8 > 6, 右移
       [3, 5, 6, 8, 4]   → 插入 6

第4步: 拿出 4, 与 8 比较
       [3, 5, 6, 8, 8]   → 8 > 4, 右移
       [3, 5, 6, 6, 8]   → 6 > 4, 右移
       [3, 5, 5, 6, 8]   → 5 > 4, 右移
       [3, 4, 5, 6, 8]   → 插入 4
```

## 7️⃣ 归并排序过程

```
         [38, 27, 43, 3, 9, 82, 10]
                /           \
        [38, 27, 43, 3]    [9, 82, 10]
          /       \          /      \
      [38, 27]  [43, 3]  [9, 82]   [10]
       /   \     /   \    /   \      |
    [38]  [27] [43] [3] [9]  [82]  [10]
       \   /     \   /    \   /      |
      [27,38]   [3,43]   [9,82]    [10]
          \       /          \      /
         [3, 27, 38, 43]    [9, 10, 82]
                \               /
            [3, 9, 10, 27, 38, 43, 82]
```

### 合并两个有序数组

```
合并 [27, 38] 和 [3, 43]:

  队列一: [27, 38]      队列二: [3, 43]
     │                       │
     └─────── 27 vs 3 ──────┘
             3 < 27 → 取 3 ✅

  队列一: [27, 38]      队列二: [43]
     │                       │
     └─────── 27 vs 43 ─────┘
             27 < 43 → 取 27 ✅

  队列一: [38]           队列二: [43]
     │                       │
     └─────── 38 vs 43 ─────┘
             38 < 43 → 取 38 ✅

  队列一: []              队列二: [43]
      → 直接追加剩余 43 ✅

  结果: [3, 27, 38, 43]
```

## 8️⃣ 二分搜索过程

```
数组: [1, 3, 4, 6, 7, 8, 9, 12]
目标: 9

第一步:
  [1, 3, 4, 6, 7, 8, 9, 12]
   L=0      mid=3       R=7
   arr[3]=6 < 9 → 搜索右半部分

第二步:
  [7, 8, 9, 12]
   L=4   mid=5   R=7
   arr[5]=8 < 9 → 搜索右半部分

第三步:
  [9, 12]
   L=6  mid=6  R=7
   arr[6]=9 == 9 ✅ 找到!
```

## 9️⃣ 各算法对比

```
        稳定  原地  时间(平均)  时间(最坏)  空间
        ────────────────────────────────────────

```
