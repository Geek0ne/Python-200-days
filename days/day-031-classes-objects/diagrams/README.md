## 1️⃣ 类与对象的关系

```
┌─────────────────────────────────────────────┐
│                 类 (Class)                   │
│                 ─────────                    │
│            class Dog:                        │
│              species = "Canis"              │ ← 类变量（共享）
│                                              │
│              def __init__(self, name):       │
│                self.name = name             │ ← 实例变量（独立）
│                                              │
│              def bark(self):                 │
│                print("Woof!")                │ ← 实例方法
└────────────┬────────────────┬───────────────┘
             │                │
     ┌───────┴──────┐  ┌────┴────────┐
     │  对象1       │  │  对象2      │
     │  ─────────   │  │  ─────────  │
     │  name="Buddy"│  │  name="Max" │
     │  species     │  │  species    │
     │  (引用类)    │  │  (引用类)   │
     └──────────────┘  └─────────────┘
```

## 2️⃣ 属性查找机制

```
obj.attribute 的查找顺序:
                   ┌──────────┐
                   │ obj.__dict__│ ← 查找实例属性
                   └─────┬────┘
                    找到? ──→ 是 ──→ 返回
                         │
                         否
                         ▼
                   ┌──────────┐
                   │ type(obj).  │
                   │ __dict__   │ ← 查找类属性
                   └─────┬────┘
                    找到? ──→ 是 ──→ 返回
                         │
                         否
                         ▼
                   ┌───────────┐
                   │ 父类链     │ ← 逐级查找父类
                   └─────┬─────┘
                    找到? ──→ 是 ──→ 返回
                         │
                         否
                         ▼
                   ┌───────────┐
                   │ Attribute │
                   │ Error     │
                   └───────────┘

示例:
  buddy = Dog("Buddy")
  buddy.name     → 先找 buddy.__dict__ → 找到 "Buddy"
  buddy.species  → 先找 buddy.__dict__ → 没找到
                  → 找 Dog.__dict__ → 找到 "Canis familiaris"

  buddy.species = "Mutt"  → 创建实例变量
                           → 不影响 Dog.species
```

## 4️⃣ `__new__` vs `__init__` 流程

```
x = MyClass(42)

    Step 1: __new__
    ┌────────────────────────┐
    │ MyClass.__new__(cls, 42)│
    │                        │
    │ 1. 分配内存             │
    │ 2. 创建空实例           │
    │ 3. 返回实例             │
    └───────────┬────────────┘
                │
                ▼ 实例已创建（属性还未设置）
                │
    Step 2: __init__
    ┌────────────────────────┐
    │ instance.__init__(42)  │
    │                        │
    │ 1. 接收已创建的实例    │
    │ 2. 初始化实例属性      │
    │    self.value = 42     │
    └───────────┬────────────┘
                │
                ▼ 实例准备好了
    x = MyClass(42)
    x.value → 42
```

## 7️⃣ `__slots__` 内存对比

```
普通类（__dict__）:
┌────────────────────────────────┐
│ 对象头      │ __dict__ → {...} │
└────────────────────────────────┘
                              │
                   ┌──────────┴─────────┐
                   │ {'x': 1, 'y': 2}    │ ← 额外字典对象
                   └────────────────────┘

__slots__ 类:
┌────────────────────────────────┐
│ 对象头      │ x=1 │ y=2        │ ← 直接存储为对象属性
└────────────────────────────────┘
  无 __dict__ → 节省内存

对比 (n=100000):
  普通: ~560 bytes/对象 → ~56 MB
  slots: ~48 bytes/对象 → ~4.8 MB
  节省: 约 91% 内存!
```
