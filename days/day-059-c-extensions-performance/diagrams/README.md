## Cython 编译流程

```
 .pyx 文件          Cython 编译器          gcc/clang           .so/.pyd
(Python+C语法)  ─────────────────→  (.c 文件)  ──────────→  (共享库)
                                        │
                                   生成 C 代码
                                   做类型优化
```

## ctypes / CFFI 调用流程

```
Python 代码        ctypes/CFFI          C 共享库          系统调用
     │               │                    │                 │
     │  创建参数      │  类型转换           │  执行 C 函数     │
     ├──────────────→├──────────────────→├────────────────→│
     │               │                    │                  │
     │  返回结果      │  类型转换回 Python  │  返回 C 值       │
     │←──────────────│←──────────────────│←─────────────────│
```

## Numba JIT 编译流程

```
Python 函数          @jit 装饰器         LLVM 编译器          机器码
   │                    │                   │                  │
   │  首次调用           │  分析函数         │  生成 LLVM IR     │
   ├───────────────────→├────────────────→├─────────────────→│
   │                    │                   │                  │
   │                    │                   │  编译为原生代码    │
   │                    │                   ├─────────────────→│
   │                    │                   │                  │
   │  后续调用           │  直接执行          │                  │
   │←────────────────────────────────────────────────────────│
```
