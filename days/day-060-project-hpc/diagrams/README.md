     ▼

## Amdahl 定律

```
加速比 = 1 / ((1-P) + P/N)

P = 可并行比例, N = 处理器数

    加速比
     10x ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ P=100%
         │                          ·····
      8x ─                    ····
         │               ····
      6x ─          ····               P=90%
         │     ····
      4x ─····                         P=70%
         │··
      2x ─·                            P=50%
         │
      1x ──────────────────────────────
         1    2    4    8   16   32  64
                 处理器数 (N)

关键洞察：
• P=50% → 64 个 CPU 最多 2x 加速
• P=90% → 64 个 CPU 最多 8.7x 加速
• P=99% → 64 个 CPU 最多 39x 加速
```
