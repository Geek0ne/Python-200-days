## 📐 每日内容结构

每个 Day 的文件夹包含：

```
days/day-XXX-topic/
├── README.md          ← 本章完整学习内容（概念 + 原理 + 图解 + 代码）
├── concepts.md        ← 核心概念与定义
├── code/              ← 代码示例目录
│   ├── example-01.py
│   ├── example-02.py
│   └── ...
├── diagrams/          ← Mermaid / ASCII 图解
│   └── ...
└── exercises/         ← 练习题
    └── ...
```

## 📝 Git 提交流程（每日）

```
1. docs(concept): 添加 [主题] 概念解释与定义
2. feat(diagram): 添加 [主题] 原理解析示意图
3. docs(example): 添加 [主题] 基础用法代码示例
4. feat(example): 添加 [主题] 实战案例代码
5. docs(exercise): 添加 [主题] 练习题与答案
6. feat: day-XXX 完整学习内容
```

**每天至少 5 次以上提交。**
